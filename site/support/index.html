支持与联系

Last updated: 2026-04-09

Cortex Terminal 是一个 relay-centric AI terminal mesh，用于连接 Mobile、Gateway 与 Worker， 并统一管理 AI terminal sessions。如果你在登录、worker pairing、连接稳定性、安装脚本或上架资料方面遇到问题，可以通过本页提供的渠道联系我们。

1. 支持范围

  • 账号登录、访问异常、worker 接入失败。
  • install token、install script 与 worker runtime 探测相关问题。
  • session relay、terminal reconnect、审计链路或兼容性问题。
  • 隐私请求、条款咨询与账号删除流程相关问题。

2. 联系方式

当前推荐通过 GitHub Issues 提交问题。请在标题中简要说明问题类型，并附上必要的上下文，例如：平台、时间、错误信息、复现步骤与相关 session / worker 标识。

3. 建议附带的信息

  • 你使用的平台与版本（iOS / Android / macOS / Web / Worker OS）。
  • 问题发生时间、是否可稳定复现、是否与特定 runtime 相关。
  • 可脱敏的日志、截图、错误消息或 install command 执行输出。
  • 如涉及账号问题，请附上可用于识别的账号标识，但不要公开提交敏感凭证。

4. 响应说明

我们会尽量按问题严重程度与可复现性进行处理，但不承诺固定响应时间或 SLA。对于安全相关问题，建议在公开提交前先最小化敏感信息暴露。

这个页面可直接作为 App Store Connect 或 Google Play Console 中的 Support URL。若你后续接入专门的支持邮箱、工单系统或企业支持门户，可在此页面继续补充。
