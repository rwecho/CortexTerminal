隐私政策

Last updated: 2026-04-09

Cortex Terminal（以下简称“本服务”）用于在 Mobile、Gateway 与 Worker 之间建立 relay-centric 的 AI terminal session。 我们重视你的隐私，并尽量以最小必要原则处理与账户、设备、会话治理和安全相关的数据。

1. 我们可能收集的数据

  • 账户与身份信息，例如登录标识、用户 ID、认证会话相关元数据。
  • 设备与连接信息，例如设备指纹、worker 标识、IP、连接时间、运行平台与基础诊断信息。
  • 服务运行数据，例如 session ID、worker pairing 记录、install token 使用记录、审计事件与错误日志。
  • 你主动提交的支持请求、反馈信息，以及与你的账号删除或隐私申请相关的沟通内容。

2. 我们如何使用这些数据

  • 提供登录、鉴权、worker 接入、session 路由与 reconnect 能力。
  • 维护系统安全，包括异常检测、滥用防护、审计追踪与故障排查。
  • 改进服务质量，例如分析错误、连接稳定性与兼容性问题。
  • 在你请求时处理支持、数据访问、更正或删除申请。

3. 终端内容与 AI 会话说明

Cortex Terminal 的设计目标是让 gateway 主要承担路由与治理职责。根据你的部署方式，某些会话元数据、审计记录或运行日志可能会被保留， 用于安全、可用性与运维目的。你通过 worker 运行的命令、输出结果以及第三方 AI provider 的处理行为，也可能受到对应 provider 或部署环境的政策约束。

4. 数据共享

除以下场景外，我们不会主动出售你的个人信息：

  • 为完成基础服务能力所必需的基础设施或身份服务提供商；
  • 为遵守法律义务、响应合法请求或保护服务安全时；
  • 在你明确授权或主动发起的集成场景中。

5. 数据保留

我们仅在实现服务目的、满足安全审计或遵守适用法律所必需的期限内保留数据。不同类型的数据保留期可能不同，例如安全日志通常会短期保存， 而账号标识与基础配置可能在账号有效期内保留。

6. 你的权利

  • 你可以请求访问、更正或删除与你相关的账户数据。
  • 你可以停止使用本服务，并按我们的账号删除流程申请删除相关账号。
  • 在适用法律允许的范围内，你还可能享有数据导出、限制处理或提出异议的权利。

7. 儿童隐私

本服务不面向儿童提供，也不以故意方式收集儿童个人信息。如果你认为有儿童在未经适当授权的情况下向我们提供了数据，请联系支持渠道。

8. 政策更新

我们可能因产品变更、法律要求或安全实践调整而更新本政策。更新后的版本会发布在本页面，并以最新更新时间为准。

9. 联系方式

如需提交隐私相关请求、支持申请或删除请求，请通过 GitHub Issues 与项目维护方联系，并在请求中注明你的账号标识、问题背景和期望处理方式。

这是一份用于 App Store / Google Play 审核与产品主页展示的公开隐私说明。若你的生产部署有额外的数据处理方式，建议在此页面基础上补充你的运营方信息与更精确的数据保留规则。
